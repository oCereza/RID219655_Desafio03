# Desafio: Construir um Gerenciador de Tarefas
Desafio feito como desafio do curso da [Escola DNC](https://www.escoladnc.com.br)

## Sobre o Projeto
Um gerenciador de tarefas simples, desenvolvido para facilitar o acompanhamento e a organização das suas atividades.

## Como acessar
Clone o repositório clicando no botão '<>Code', ou acesse diretamente pelo link:
